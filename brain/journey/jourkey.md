# Journey

My software development journey started with curiosity about how applications work.

Initially, I focused on programming fundamentals and Java.

As I learned more, I became interested in backend development because I enjoyed building systems that power applications.

Over time, I expanded into full-stack development, learning React, Node.js, databases, and cloud deployment.

Leadership opportunities allowed me to develop communication, management, and organizational skills while working with student communities.

Today, my focus is on building impactful products using backend technologies and AI while continuously improving as a software engineer.
